# tech stack identifier

<p>This is chrome extension that is able to detect the tech stack of the webpages</p>
# <b>Currecntly it can detect the following</b>

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>Javascript</li>
  <li>jquery</li>
  <li>php</li>
  <li>Angular</li>
  <li>React</li>
  <li>Bootstrap</li>
  <li>Google fonts api</li>
  
</ul>
